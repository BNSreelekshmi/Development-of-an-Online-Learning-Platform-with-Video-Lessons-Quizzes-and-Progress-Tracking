# Online Learning Platform

## Project Description
This project is a web-based online learning platform developed using Python (Flask) and PostgreSQL. It allows students to access video lessons, take quizzes, and track their learning progress.

##  Features
- User-friendly interface
- Video-based learning
- Course listing
- Quiz functionality
- Progress tracking (basic)
- Backend with Flask
- PostgreSQL database integration

##  Technologies Used
- Frontend: HTML, CSS, Bootstrap
- Backend: Python (Flask)
- Database: PostgreSQL
- Tools: VS Code, GitHub
